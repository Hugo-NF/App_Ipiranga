Detailed Description

Class Search
Responsável por definir a interface de buscas entre a view e a DB.
Define atributos que serão parametros de busca, bem como os metodos de busca.

Note
: Todas os metodos dessa classe realizam operações atômicas, consistentes, isoladas e duráveis(ACID). Propriedade herdada do sqlite.
