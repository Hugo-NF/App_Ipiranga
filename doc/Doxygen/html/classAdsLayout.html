Detailed Description

Class SerachFriends
Classe responsável por criar a interface grafica para exibição de informações de um unico anuncio. Permitindo excluir o anuncio em qualquer hora. Esta classe esta diretamente conectada a classe com respectivo nome acrescido de -ui, a qual é responsável pela criação de muitos widgets exibidos ao usuario.
classe herda de -ui todos seus atributos e metodos.
