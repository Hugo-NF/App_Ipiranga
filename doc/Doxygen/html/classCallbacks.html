Detailed Description

Class Callbacks
Classe responsável por definir métodos estáticos para mapeamento dos retornos da DB dentro das suas classes.
Todas os metodos dessa classe realizam operações atômicas, consistentes, isoladas e duráveis(ACID). Propriedade herdada do sqlite.
