Detailed Description

Class Historic
Classe responsável por definir os atributos de uma entrada de histórico, bem como metodos de inserção e remoção

Note
: Todas os metodos dessa classe realizam operações atômicas, consistentes, isoladas e duráveis(ACID). Propriedade herdada do sqlite.

Metodo para a recuperação do historico de um usuario

  • 1: inteiro com o id do usuário
  • 2: booleano indicando a requisição do historico como vendedor
