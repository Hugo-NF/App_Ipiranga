Detailed Description

Class Ads :Todas os metodos dessa classe realizam operações atômicas, consistentes, isoladas e duráveis(ACID).Propriedade herdada do sqlite.
Responsável por definir os atributos de um anuncio e definir as operações de inserção e atualização de um anuncio.
