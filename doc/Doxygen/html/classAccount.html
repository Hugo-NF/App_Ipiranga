Detailed Description

Class Account Classe responsável por definir métodos estaticos para manipulação das contas de usuário : Todas os metodos dessa classe realizam operações atômicas, consistentes, isoladas e duráveis(ACID).Propriedade herdada do sqlite. Define a interface para manipulação de usuário com a camada View
